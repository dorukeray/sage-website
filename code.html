Code

Code using Sage.

Because Sage is an approach for data interactions; there are libraries, tools and other resources to help you get started working with Sage.

Reference Implementations

Server

PHP

Sage.php

  • GitHub: dorkodu/sage.php

  • Status: Work In Progress

  • License: MIT

The reference implementation of the Sage protocol, in PHP.

This library is a thin middleware on top of your existing data and business logic. It provides the utility for creating an awesome Sage API for your app.

  • A Type System to express your app’s data.
  • Validation and Introspection of this Type System
  • Parsing, validating and executing Sage queries against this Type System
  • Rich error reporting, including query validation and execution errors

Client

JavaScript

Sage.js

  • GitHub: dorkodu/sage.js

  • Status: Work In Progress

  • License: MIT

Sage.js is the simple, lightweight and awesome Sage client for JS.


Sage.js enables you to manage your both local & remote data, and structure your code in an economical, predictable, and effective way.

  • Declarative & efficient data exchange with Sage.
  • Manage your app's data requirements.
  • Universally compatible.
  • Isomorphic.
  • No dependencies, just vanilla JavaScript.
